# Crossmint (crossmint)

Crossmint is a Web3 platform offering APIs for wallets, NFT minting, checkout, payments, embedded checkout, and verifiable credentials. Supports server-managed wallets across EVM and Solana, fiat-on-ramp checkout, and credit-card-funded NFT purchases.

Crossmint does not publish a public OpenAPI spec download URL at this time. Endpoints are documented at https://docs.crossmint.com.

## APIs
- **Crossmint Wallets API** - REST API to create and manage server-side custodial and non-custodial smart wallets across EVM and Solana, sign transactions, manage delegated permissions.
- **Crossmint Minting API** - REST API to mint and manage NFTs on multiple chains, including collection creation, batch minting, and metadata management.
- **Crossmint Checkout API** - REST API for hosted and headless checkout flows allowing credit-card and crypto purchases of NFTs.
- **Crossmint Verifiable Credentials API** - REST API to issue, manage, and verify verifiable credentials anchored on chain.
- **Crossmint Headless Checkout (Order API)** - REST Order API for fully headless on-chain commerce flows including fiat and crypto payment intents.
